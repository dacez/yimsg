# yimsg 产品基础信息（全部客服共享）

- **形态**：yimsg 是可私有部署的即时通讯系统，一次部署后既可作为独立 Web IM 使用，也可以用 UIKit 一行代码嵌入到其他网站或后台系统。
- **协议**：客户端与服务端之间走 WebSocket 二进制帧协议；文件/图片上传、静态资源和媒体访问走 HTTP。
- **多端登录**：同一账号可以同时在多台设备、多个浏览器登录，互不挤下线；每次登录生成独立的 session token，默认有效期 30 天。
- **消息与会话保留**：不是按时间保留，而是按数量：每个用户最多保留 100000 条消息、10000 个会话，超出部分由后台任务自动清理（部署方可调整）。
- **上传大小限制**：头像最大 5MB，图片最大 10MB，普通文件最大 100MB（部署方可在 config.toml 里调整）。
- **拉黑与屏蔽**：支持把某个用户加入黑名单（拉黑），也支持对某个会话开启消息免打扰（屏蔽/静音），两者互不影响、可分别使用。
- **群聊**：目前没有固定的群人数硬上限，实际人数由服务器资源和部署方配置决定。
