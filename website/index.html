Deployment

四种部署方式，总有一款适合你

无论网络环境如何，一台机器、几分钟即可把服务跑起来——数据始终留在你的办公室。

A

有域名 + 公网 IP

直接部署，客户端通过你自己的域名直连服务，链路最短、最纯粹。

B

无域名 · 有公网 IP

由 yimsg.im 提供路由，将访问引导到你的公网 IP，无需自备域名。

C

无域名 · 无公网 IP

由 yimsg.im 做中转，内网机器也能对外提供服务。

D

内网盒子 · 即插即用

在路由器旁接入一台迷你主机大小的小盒子，即可为整个内网提供即时通讯，无需公网、无需复杂配置，开箱即用。

四种方式下，数据都只留在你办公室里的机器或盒子里；yimsg.im 仅在需要时负责连通，从不保存你的消息。
