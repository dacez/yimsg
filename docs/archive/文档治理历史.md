## 〇、2026-07-16 文档架构重组记录

| 变更 | 处理结果 | 原因 |
|---|---|---|
| `docs/接口总览.md` | 移入 `docs/protocol/接口总览.md`，删除与 `sdk接口说明.md` / `UIKit方案.md` 重复的 UIKit / SDK 接口详解与使用示例（约 450 行），收敛为「规范 + SDK ↔ 服务端映射 + 服务端接口矩阵」 | `protocol/` 收拢所有对外契约文档；接口详解只在各层权威文档维护一份 |

| `server/消息撤回设计方案.md`、`server/消息扩展设计方案.md` | 合并到 [`../server/消息能力方案.md`](../server/消息能力方案.md) | 撤回基于 `MSG_TYPE_EXT` envelope 与消息同步链路，应与扩展消息协议在同一事实源维护 |
